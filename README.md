# Sistema de Gestão de Transportadora 🚚📦

Este repositório contém um projeto de **CRUD (Create, Read, Update, Delete)** desenvolvido em **Java** para gerenciar as operações de uma transportadora que realiza entregas interestaduais. O sistema permite gerenciar informações de entregas, motoristas, veículos e clientes de forma eficiente.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é:
- Desenvolver um sistema simples de gestão para transportadoras.
- Consolidar o uso de **Java** para construção de aplicações CRUD.
- Praticar conceitos como **orientação a objetos (POO)**, manipulação de dados e persistência em banco de dados.

---

## 🚀 Funcionalidades

- **Cadastro de Entregas:** Adicione informações sobre entregas interestaduais.
- **Gerenciamento de Clientes:** Insira, visualize, edite e exclua registros de clientes.
- **Gestão de Motoristas e Veículos:** Organize dados de motoristas e veículos associados às entregas.
  
---
